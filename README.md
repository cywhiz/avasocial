# AvaSocial – AI Yoga Trainer

**AvaSocial** is a premium web-based AI yoga assistant that uses your webcam to provide real-time pose analysis. Designed with a focus on high-end aesthetics and user experience, it helps you master yoga poses by tracking your posture and guiding you through timed sessions using advanced machine learning.

## 🚀 Installation

To run AvaSocial locally, you need a web server to handle the machine learning models correctly.

1. **Clone the project:**
   ```bash
   git clone https://github.com/your-username/avasocial.git
   cd avasocial
   ```

2. **Run a local server:**
   - If you have **Python**: `python -m http.server 8000`
   - If you use **VS Code**: Use the "Live Server" extension.

3. **Open the app:**
   Go to `http://localhost:8000` in your browser.

## 🧘 How to Use

1. **Grant Camera Access:** Click "Allow" when your browser asks for camera permissions.
2. **Start Session:** Click the "Start Yoga Session" button.
3. **Strike a Pose:** Match the yoga pose shown on the screen. The AI will automatically detect when you are in the correct position.
4. **Hold & Complete:** Hold each pose for 10 seconds. The timer will change color from Red to Green as you approach completion. Finish all poses to complete your session!
